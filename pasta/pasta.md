# 義大利麵

這個食譜分成三個部分:

1. 麵條
2. 醬汁
3. 蛋白質 (肉類、海鮮)

醬汁的食譜請見[醬汁](./sauce/)，肉類、海鮮的食譜請見[蛋白質](./protein/)。

以下是煮麵條的方法。

1. 煮麵鍋放一些鹽加水煮滾。鹽的量隨便。可以先加多一點 (比泡麵調味包還多的量)，如果覺得太鹹下次就放少一點。
2. 放麵條並且開始計時。如果麵條沒辦法完全泡進水裡的話就等一下下再用夾子把麵條壓進去。
3. 麵條的包裝上會寫需要煮多久，時間到之前兩分鐘就可以先夾一條起來吃吃看硬度。持續試吃直到麵條變成你喜歡的硬度。[^1]
4. 把麵條夾到醬汁裡拌一拌。

煮麵條和煮醬汁會同時進行。我會確保麵條煮好的時候醬汁已經煮好了，這樣就可以直接把麵條夾進醬汁裡，比較方便。

**待補: timing**

[^1]: 蒜頭橄欖油義大利麵的作法比較不一樣，麵條煮到一半熟就要夾起來放到醬汁裡面繼續煮，請見[蒜頭橄欖油醬汁](./sauce/)。
