# 義大利麵

[回首頁](../README.md)

煮義大利麵分成三個部分:

1. 麵條
2. [醬汁](./sauce/sauce.md)
3. [蛋白質](./protein/protein.md) ( 肉類、海鮮 )

流程大概是這樣:[^1]

1. 用平底鍋煮蛋白質 ( 肉類、海鮮 )。
2. 用煮麵鍋煮麵的同時用平底鍋煮醬汁。
3. 麵煮熟後夾進醬汁裡攪一攪。

通常煮醬汁的時間比煮麵的時間短一些，所以如果要加快速度的話可以在蛋白質煮到一半的時候就開始煮麵。  
醬汁的食譜請見[醬汁](./sauce/sauce.md)，肉類、海鮮的食譜請見[蛋白質](./protein/protein.md)。

以下是煮麵條的方法。我一餐大概吃 75 克的麵條。

1. 煮麵鍋放一些鹽加水煮滾。鹽的量隨便。可以先加多一點 (比泡麵調味包還多的量)，如果覺得太鹹下次就放少一點。
2. 放麵條並且開始計時。如果麵條沒辦法完全泡進水裡的話就等一下下等麵變軟再用夾子壓進去。
3. 麵條的包裝上會寫需要煮多久，時間到之前兩分鐘就可以先夾一條起來吃吃看硬度。持續試吃直到麵條變成你喜歡的硬度。
4. 把麵條夾到醬汁裡拌一拌。

[^1]: 有幾個例外: [蒜頭橄欖油義大利麵](./sauce/garlic-olive-oil.md)是麵條煮到半熟就加進醬汁裡繼續煮熟；[培根蛋義大利麵](./sauce/carbonara.md)是煮麵的同時煎培根，麵煮熟後跟培根攪一攪再加入醬汁。
