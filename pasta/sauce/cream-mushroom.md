# 奶油蘑菇醬汁

[回上頁](./sauce.md)

## 食材 ( 1 人份 )

- 蘑菇 一小碗
- 重鮮奶油 ( heavy cream ) 50 毫升
- 無鹽奶油 10 克
- 帕馬森起司 ( Parmigiano Reggiano ) 20 克
- 蒜頭 1~2 瓣
- 黑胡椒

## 作法

1. 蘑菇切片，蒜頭切末。
2. 平底鍋放點橄欖油預熱，炒磨菇。
3. 放鹽、黑胡椒、蒜頭一起炒。
4. 蘑菇炒一炒會放出很多水分，要炒久一點把水分蒸乾一些，避免醬汁太稀。
5. 轉小火，放入無鹽奶油攪一攪。
6. 放入重鮮奶油和帕馬森起司攪一攪。
7. 把煮好的麵條夾進來攪一攪。如果醬汁太稠可以加一些煮麵水。

## 提醒

- 加入奶油後要隨時注意火力。火太大容易油水分離 (看起來有點結塊、分離出透明液體)。建議轉到最小火或偶爾關火。

## 推薦組合

### 麵條
- 直麵 ( spaghetti )
- 細扁麵 ( linguine )
- 水管麵 ( rigatoni )
- 筆管麵 ( penne rigate )

### 蛋白質
- [牛排](../protein/steak.md)
- 鮭魚
- [雞肉](../protein/chicken.md)
- 蝦子干貝
