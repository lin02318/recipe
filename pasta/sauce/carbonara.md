# 培根蛋義大利麵

[回上頁](./sauce.md)

## 食材 ( 1 人份 )

- 培根 一些[^1]
- 雞蛋 2 顆
- 羊奶起司 ( Pecorino Romano ) 10 克[^2]
- 帕馬森起司 ( Parmigiano Reggiano ) 20 克
- 粗的黑胡椒

## 作法[^3]

1. 把一顆全蛋、一顆蛋黃、兩種起司、黑胡椒混合拌勻。
2. 培根切塊，放點橄欖油在平底鍋用中火加熱。橄欖油放一點點就好，因為培根加熱之後會跑出很多油。
3. 開始煮麵條。
4. 培根開始出現一點焦焦的顏色、開始冒小泡泡的時候關火。如果覺得煎培根跑出來的油太多的話可以撈掉一些。
5. 把煮好的麵條夾進來攪一攪。大概攪 30 秒讓麵冷卻一點。
6. 把步驟一的醬汁倒進去攪一攪。如果醬汁太稠可以加一些煮麵水。

## 提醒

- 為了避免倒醬汁的時候平底鍋的溫度太高，理想的狀況是麵煮好前 3~5 分鐘就把煎培根的平底鍋關火冷卻。
- 如果醬汁攪一攪變成蛋花代表麵的溫度太高，下次就等久一點再倒醬汁。

## 推薦組合

### 麵條
- 直麵 ( spaghetti )
- 細扁麵 ( linguine )

### 蛋白質
就是培根。我沒有加過其他東西。

[^1]: 如果買得到 guanciale 或 pancetta 也可以試試看。
[^2]: 兩種起司的比例可以自己調整。羊奶起司很鹹所以我加少一點。
[^3]: 培根蛋義大利麵的醬汁不用煮，所以煮麵的同時煎培根，麵煮熟後跟培根攪一攪再加入醬汁。
