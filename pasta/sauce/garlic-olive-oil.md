# 蒜頭橄欖油醬汁

## 食材 ( 1 人份 )

- 蒜頭 1 顆
- 橄欖油
- 花椰菜 幾朵
- 巴西里碎
- 辣椒碎 ( optional )

## 作法

1. 花椰菜洗乾淨，切成適合的大小。蒜頭切薄片。
2. 蒜頭和橄欖油放入平底鍋。橄欖油可以放多一點，讓全部的蒜頭都完整接觸到橄欖油。
3. 中小火加熱，讓蒜頭有一點冒泡泡但不要太多。如果要加辣椒碎的話在這時候加。
4. 把花椰菜放到煮麵鍋裡煮兩分鐘。
5. 一旦有蒜頭開始變色，關火並且撈一杓煮麵水到醬汁裡再開火。[^1]
6. 把花椰菜和麵條放到醬汁裡煮，加一些煮麵水淹過大概一半的麵。
7. 火開大一點，把麵繼續煮熟的同時蒸乾水分。如果水太少就再加一些煮麵水。
8. 持續試吃麵條直到麵條變成你喜歡的硬度，關火並加入巴西里碎攪一攪。

## 提醒

- 麵條放入醬汁繼續煮熟的時候要同時注意麵條的熟度和醬汁的水量，理想的狀況是麵條熟的時候醬汁的水分也蒸乾到適合的稠度。為了避免麵條已經煮熟了醬汁還是太水，建議一次加一點煮麵水就好，不夠再加。

## 推薦組合

### 麵條
- 直麵 ( spaghetti )
- 細扁麵 ( linguine )

### 蛋白質
- [牛排](../protein/steak.md)
- 雞肉

[^1]: 大部分的醬汁都是跟麵條完全分開煮，但是蒜頭橄欖油義大利麵是煮麵條煮到半熟就加進醬汁裡繼續煮熟。如果不確定怎麼拿捏時間的話建議同時開始煮麵條和煮醬汁，這樣麵條煮到一半時間的時候大概醬汁就大概可以加煮麵水了。
